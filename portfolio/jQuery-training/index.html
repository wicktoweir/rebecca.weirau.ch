jQuery - The Basics

  • jQuery is a JavaScript library that makes working with webpages easier.
  • It is a way to make your webpages more interesting and interactive with animations and dynamic content.

Making It Work

  • jQuery must be linked to your HTML document in order to be used.
  • To do this, we need to reference the jQuery library in much the same way as we reference CSS files, by linking it in the HEAD of the HTML document.
  • Look at Example 1 of the handout for how to link to jQuery.

SlideUp and SlideDown

  • The slideUp and slideDown functions are seen in many websites in the form of a drop down menu.
  • This is a simplified version of what a drop down menu does.
Click Here!
Hello

How Did We Do That?

  • We use two div's, the first is the one with "Click Here!" and the second is the one with "Hello".
  • The "Hello" div is initially styled to be invisible.
  • When we click on the "Click Here!" div, we tell jQuery to make the "Hello" div visible using a slide down effect.
  • Finally, when we move the mouse out of the "Click Here!" div, we tell jQuery to slide the "Hello" div back up and make it invisible.
  • Example 4 has more on this trick!
